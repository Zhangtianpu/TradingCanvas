# TradingCanvas 需求文档

## 一、产品定位

面向A股超短交易者（打板为主）的题材情绪复盘工具，帮助追踪 **"题材 → 龙头/中军 → 情绪周期"** 主线，纯前端部署于 GitHub Pages，数据存储于 localStorage。

## 二、目标用户

- A股超短线交易者，以打板（涨停板买入）为主要买入方式
- 需要每日复盘题材轮动、龙头地位、情绪周期位置
- 习惯手动记录复盘信息，需要结构化工具辅助

## 三、核心功能模块

### 3.1 题材管理

| 功能 | 说明 |
|------|------|
| 创建题材 | 输入题材名称、所属板块、首次爆发日期、题材等级（主线/支线/轮动） |
| 题材列表 | 按日期倒序展示，支持按等级/状态筛选 |
| 题材详情 | 查看题材下的个股、情绪周期、关键事件 |
| 题材状态 | 爆发 → 发酵 → 高潮 → 分歧 → 退潮 → 调整，手动标记 |
| 关联个股 | 每个题材下可添加龙头、中军、跟风个股 |

### 3.2 个股复盘

| 功能 | 说明 |
|------|------|
| 添加个股 | 股票代码、名称、所属题材、角色（龙头/中军/跟风/补涨） |
| 打板记录 | 记录打板日期、涨停价、封板时间、封单量、是否炸板 |
| 连板统计 | 自动计算连板天数、断板记录 |
| 个股标签 | 一字板、T字板、换手板、地天板等标签 |
| 买卖记录 | 记录买入/卖出日期、价格、数量、盈亏 |

### 3.3 情绪周期

| 功能 | 说明 |
|------|------|
| 周期标记 | 标记当前市场情绪阶段：冰点→修复→升温→高潮→退潮 |
| 每日情绪 | 记录当日涨停数、跌停数、连板高度、炸板率 |
| 情绪曲线 | 可视化展示情绪指标趋势 |
| 周期关联 | 标记当前周期对应的龙头题材 |

### 3.4 每日复盘

| 功能 | 说明 |
|------|------|
| 日期选择 | 选择复盘日期，默认当天 |
| 复盘摘要 | 自由文本记录当日市场整体感受 |
| 题材梳理 | 当日各题材表现、龙头表现 |
| 操作反思 | 记录当日操作、得失、改进点 |
| 明日计划 | 记录明日关注方向、目标个股 |

### 3.5 数据管理

| 功能 | 说明 |
|------|------|
| 自动保存 | 所有操作自动保存到 localStorage |
| 导出备份 | 一键导出全部数据为 JSON 文件 |
| 导入恢复 | 从 JSON 文件导入恢复数据 |
| 数据清空 | 支持清空所有数据（需二次确认） |

## 四、页面结构

```
├── 首页/看板（今日概览）
├── 题材管理
│   ├── 题材列表
│   └── 题材详情（含关联个股）
├── 个股复盘
│   ├── 个股列表
│   └── 个股详情（含打板/买卖记录）
├── 情绪周期
│   ├── 情绪指标录入
│   └── 情绪曲线图
├── 每日复盘
│   └── 按日期的复盘记录
└── 设置（数据导入导出）
```

## 五、非功能需求

- 纯静态页面，可部署到 GitHub Pages
- 无后端依赖，所有数据存储在浏览器 localStorage
- 移动端友好，支持手机浏览器使用
- 快速响应，操作延迟 < 200ms
- 数据安全：导入前自动备份，清空前二次确认
