# TradingCanvas UI 设计文档

## 一、设计风格

- **暗色主题为主**：交易者习惯深色界面，减少视觉疲劳
- **配色方案**：
  - 背景：#0d1117（深黑蓝）
  - 卡片背景：#161b22
  - 边框：#30363d
  - 主文字：#e6edf3
  - 次要文字：#8b949e
  - 涨/红：#f85149
  - 跌/绿：#3fb950
  - 强调色/蓝：#58a6ff
  - 警告/黄：#d29922
  - 龙头/金：#f0c040

- **字体**：系统默认等宽字体，数字对齐
- **圆角**：8px 卡片圆角，4px 按钮圆角
- **间距**：8px 基础单位

## 二、布局结构

### 2.1 整体布局

```
┌─────────────────────────────────────────┐
│  顶部导航栏（Logo + 导航 + 日期）        │
├────────┬────────────────────────────────┤
│        │                                │
│  侧边  │      主内容区                  │
│  导航  │                                │
│        │                                │
│        │                                │
├────────┴────────────────────────────────┤
│  底部信息栏（移动端导航）                 │
└─────────────────────────────────────────┘
```

- 桌面端：左侧固定导航 + 右侧内容区
- 移动端：底部 Tab 导航 + 全屏内容区

## 三、交互规范

- 所有列表支持滑动删除（移动端）和右键菜单（桌面端）
- 表单输入即时保存，无需手动点击保存按钮（复盘页除外）
- 日期选择器默认当天
- 下拉选择支持搜索过滤
- 操作成功/失败使用 Toast 提示，3秒自动消失
- 删除操作需二次确认弹窗
